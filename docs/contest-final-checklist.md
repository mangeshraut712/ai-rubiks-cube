# Final Contest Checklist

Use this before clicking Submit on Devpost.

## Required

- [ ] Public GitHub repository URL is final and accessible
- [ ] README has local setup + deployment instructions
- [ ] Backend deployed on Google Cloud Run
- [ ] Cloud Run service URL added to Devpost submission
- [ ] Frontend public URL added to Devpost submission
- [ ] Architecture diagram uploaded to Devpost media
- [ ] <4 minute demo video uploaded and linked
- [ ] Google Cloud proof included:
  - [ ] short recording from GCP console, or
  - [ ] direct repo links to `terraform/main.tf`, `cloudbuild.yaml`, `deploy.sh`

## Live Agents Category Fit

- [ ] Real-time audio interaction shown in demo
- [ ] Real-time visual understanding shown in demo (webcam/cube)
- [ ] Interruption/barge-in handling shown in demo
- [ ] Gemini model usage explicitly mentioned in description
- [ ] Google Cloud hosting explicitly mentioned in description

## Optional Bonus

- [ ] Public GDG profile link added
- [ ] Optional build-content link added (blog/podcast/video with challenge mention)
- [ ] Social post includes hashtag `#GeminiLiveAgentChallenge`
