🧠 Algorithm Guide

BFS (Bidirectional) Recommended

Searches from both start and goal. Fast and reliable for any scramble length. Always finds optimal solution.

A* (Heuristic Search)

Uses heuristics to guide search. Good for short scrambles (<15 moves). May timeout on complex scrambles.

IDA* (Memory Efficient)

Iterative deepening with heuristics. Uses minimal memory. Best for constrained environments.

🔤 Move Notation

U', D', L', R', F', B' = Counter-clockwise (prime moves)
