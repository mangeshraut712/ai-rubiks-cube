# Devpost Submission Pack

This folder contains the judge-facing submission material for the Gemini Live Agent Challenge entry.

- `project-description.md`
  Copy-ready project summary, features, stack, learnings, and value proposition.
- `requirements-crosscheck.md`
  Requirement-by-requirement mapping against the current Devpost brief.

  Proof that the app backend is deployed on Google Cloud and where that is represented in code.

  Upload-ready architecture diagram.
- `architecture-notes.md`

## Project framing

This repo contains two product surfaces, but one submission:

- Part 1: `Gemini Live Tutor`
- Part 2: `Cubey Core 2x2 Lab`

The submission is for the combined experience, with Part 1 as the primary `Live Agents` category fit and Part 2 as the transparent core-lab companion that demonstrates solving logic, deterministic search, and playback.
